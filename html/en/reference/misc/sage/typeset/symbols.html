class sage.typeset.symbols.CompoundSymbol(character, top, extension, bottom, middle=None, middle_top=None, middle_bottom=None, top_2=None, bottom_2=None)Â¶

Bases: sage.structure.sage_object.SageObject

A multi-character (ascii/unicode art) symbol

INPUT:

Instead of string, each of these can be unicode in Python 2:

  • character â string. The single-line version of the symbol.

  • top â string. The top line of a multi-line symbol.

  • extension â string. The extension line of a multi-line symbol (will be repeated).

  • bottom â string. The bottom line of a multi-line symbol.

  • middle â optional string. The middle part, for example in curly braces. Will be used only once for the symbol, and only if its height is odd.

  • middle_top â optional string. The upper half of the 2-line middle part if the height of the symbol is even. Will be used only once for the symbol.

  • middle_bottom â optional string. The lower half of the 2-line middle part if the height of the symbol is even. Will be used only once for the symbol.

  • top_2 â optional string. The upper half of a 2-line symbol.

  • bottom_2 â optional string. The lower half of a 2-line symbol.
