Features for testing whether interpreter interfaces are functional¶

class sage.features.interfaces.InterfaceFeature(name, module, description)¶

Bases: sage.features.Feature

A Feature describing whether an Interface is present and functional.

class sage.features.interfaces.Macaulay2(name, module, description)¶

Bases: sage.features.interfaces.InterfaceFeature

A Feature describing whether sage.interfaces.macaulay2.Macaulay2 is present and functional.

EXAMPLES:

sage: from sage.features.interfaces import Macaulay2
sage: Macaulay2().is_present()  # random
FeatureTestResult('macaulay2', False)
