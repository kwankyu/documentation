graphs¶

Description¶

A database of graphs. Created by Emily Kirkman based on the work of Jason Grout. Since April 2012 it also contains the ISGCI graph database.

Upstream Contact¶

  • For ISGCI:

    H.N. de Ridder (hnridder@graphclasses.org)

  • For Andries Brouwer’s database:

    The data is taken from from Andries E. Brouwer’s website (https://www.win.tue.nl/~aeb/). Anything related to the data should be reported to him directly (aeb@cwi.nl)

    The code used to parse the data and create the .json file is available at https://github.com/nathanncohen/strongly_regular_graphs_database.
