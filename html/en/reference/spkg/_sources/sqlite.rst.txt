.. _spkg_sqlite:

sqlite: An SQL database engine
==============================

Description
-----------

SQLite is a software library that implements a self-contained,
serverless, zero-configuration, transactional SQL database engine.

License
-------

Public Domain


Upstream contact
----------------

-  https://www.sqlite.org

Dependencies
------------

-  readline


Special Update/Build Instructions
---------------------------------

-  Use the autoconf version of sqlite.

Type
----

standard

-------------------

Equivalent System Packages
--------------------------

arch:

.. CODE-BLOCK:: bash

    $ sudo pacman -S  sqlite-devel

conda:

.. CODE-BLOCK:: bash

    $ conda install sqlite-devel

cygwin:

.. CODE-BLOCK:: bash

    $ apt-cyg install sqlite-devel

Debian/Ubuntu:

.. CODE-BLOCK:: bash

    $ sudo apt-get install  sqlite-devel

Fedora/Redhat/CentOS:

.. CODE-BLOCK:: bash

    $ sudo yum install  sqlite-devel

freebsd:

.. CODE-BLOCK:: bash

    $ sudo pkg install sqlite-devel

gentoo:

.. CODE-BLOCK:: bash

    $ sudo emerge sqlite-devel

homebrew:

.. CODE-BLOCK:: bash

    $ brew install sqlite-devel

macports:
install the following packages: sqlite-devel
nix:

.. CODE-BLOCK:: bash

    $ nix-env --install sqlite-devel

opensuse:

.. CODE-BLOCK:: bash

    $ sudo zypper install sqlite-devel

slackware:

.. CODE-BLOCK:: bash

void:

.. CODE-BLOCK:: bash
