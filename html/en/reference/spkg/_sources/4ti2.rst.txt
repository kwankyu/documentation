4ti2: Algebraic, geometric and combinatorial problems on linear spaces
======================================================================

Description
-----------

A software package for algebraic, geometric and combinatorial problems
on linear spaces. Available at www.4ti2.de.

License
-------

Equivalent System Packages
--------------------------

arch:

.. CODE-BLOCK:: bash

    $ sudo pacman -S  4ti2


conda:

.. CODE-BLOCK:: bash

    $ conda install 4ti2


cygwin:

.. CODE-BLOCK:: bash

    $ apt-cyg install lib4ti2_0 lib4ti2-devel


Debian/Ubuntu:

.. CODE-BLOCK:: bash

    $ sudo apt-get install  4ti2


Fedora/Redhat/CentOS:

.. CODE-BLOCK:: bash

    $ sudo yum install  4ti2


freebsd:

.. CODE-BLOCK:: bash

    $ sudo pkg install math/4ti2


gentoo:

.. CODE-BLOCK:: bash

    $ sudo emerge sci-mathematics/4ti2


opensuse:

.. CODE-BLOCK:: bash

    $ sudo zypper install 4ti2 4ti2-devel


See https://repology.org/project/4ti2/versions


If the system package is installed, ./configure will check whether it can be used.
