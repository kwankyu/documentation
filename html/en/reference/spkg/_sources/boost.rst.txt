.. _spkg_boost:

boost: Portable C++ libraries (full set)
========================================

Description
-----------

Boost provides free peer-reviewed portable C++ source libraries.

License
-------

Boost software license (GPL compatible)


Upstream Contact
----------------

Home page: http://boost.org

Dependencies
------------

None

Type
----

optional


Version Information
-------------------

package-version.txt::

    1_66_0


Equivalent System Packages
--------------------------

arch:

.. CODE-BLOCK:: bash

    $ sudo pacman -S  boost-devel

conda:

.. CODE-BLOCK:: bash

    $ conda install boost-devel

cygwin:

.. CODE-BLOCK:: bash

    $ apt-cyg install boost-devel

Debian/Ubuntu:

.. CODE-BLOCK:: bash

    $ sudo apt-get install  boost-devel

Fedora/Redhat/CentOS:

.. CODE-BLOCK:: bash

    $ sudo yum install  boost-devel

freebsd:

.. CODE-BLOCK:: bash

    $ sudo pkg install boost-devel

homebrew:

.. CODE-BLOCK:: bash

    $ brew install boost-devel

macports:
install the following packages: boost-devel
nix:

.. CODE-BLOCK:: bash

    $ nix-env --install boost-devel

opensuse:

.. CODE-BLOCK:: bash

    $ sudo zypper install boost-devel

slackware:

.. CODE-BLOCK:: bash

    $ sudo slackpkg install boost-devel

void:

.. CODE-BLOCK:: bash

    $ sudo xbps-install  boost-devel

See https://repology.org/project/boost-devel/versions

If the system package is installed, ./configure will check whether it can be used.
